# Unity-Boilerplate

###A boilerplate to any Unity game projects

##Requirements
* [Unity 5.0.1f1](http://unity3d.com/get-unity)

##Features
* Logo Intro
* Menu
* Cast
* Select Scenes (TODO)
* Options (TODO)
* Quit
* GameManager
* Pause System
* Mini State Machine

##Folder Structure
````
|-- Resources
  |-- Animations
    |-- clips
    |-- controllers
  |-- Audio
  |-- Fonts
  |-- Images
    |-- gui
    |-- textures
  |-- Materials
  |-- Models
  |-- Prefabs
|-- Scenes
|-- Script
  |-- Controllers
  |-- GameManager
  |-- GUI
```
